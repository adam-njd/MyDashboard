# Mine Portfolio

A modern, interactive personal portfolio website with customizable themes and settings.

## 🎨 Features

- **Theme Customization** - Switch between multiple color schemes
- **Dynamic Background** - Toggle animated backgrounds on/off
- **Interactive Bullets** - Show/hide UI bullet points
- **Responsive Design** - Mobile-friendly layout
- **Fast & Lightweight** - Pure HTML, CSS, and JavaScript (no dependencies)
- **FontAwesome Icons** - Professional icon library included
- **Google Fonts** - Premium typography with Open Sans

## 📁 Project Structure


## 🚀 Getting Started

### Local Preview
1. Clone or download this repository
2. Open `index.html` in your web browser
3. Use the settings gear icon to customize colors and effects

### Live Demo
Deploy to GitHub Pages:
1. Push this repo to GitHub
2. Go to **Settings** → **Pages**
3. Select `main` branch from the source dropdown
4. Save and visit your live site

## ⚙️ Customization

### Change Colors
Click the settings gear icon in the top corner and select from 4 preset color schemes:
- Red
- Teal
- Purple
- Green

### Toggle Features
- **Change Background** - Enable/disable animated backgrounds
- **Show Bullets** - Toggle visual bullet markers on/off

### Edit Content
- Modify `index.html` to change text, images, and structure
- Update `CSS/master.css` for styling adjustments
- Extend `main.js` for additional interactivity

## 📦 Dependencies

- **FontAwesome 7.1.0** - Icon library (included locally)
